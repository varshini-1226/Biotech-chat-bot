# Biotech Q&A Chatbot

## project overview

The Biotech Q&A Chatbot is an AI-powered web application designed to help students understand biotechnology concepts easily.

It provides instant answers using a hybrid approach combining:
- Local FAQ Database (SQLite)
- MeSH (NCBI) biomedical definitions
- AI-generated responses (Groq LLaMA model)

---

## Objectives
- Provide quick and accurate biotechnology answers
- Simplify complex concepts for students
- Combine database + AI for better results
- Build an interactive chatbot interface

---

## Technologies Used
- Python
- Streamlit (Frontend UI)
- SQLite (Database)
- Groq API (LLaMA 3.1)
- Biopython (NCBI MeSH)
- Dotenv (.env for API keys)

---

## Features
- Chat-style interface
- Instant responses
- FAQ-based answers
- AI-generated explanations
- Biomedical term definitions (MeSH)
- Web-based application
